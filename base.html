Base - The base plug for parader¶

(plugs.base.preload info)¶

Adds the default bookmarks and context buttons and displays the Preloading... message.

(plugs.base.onload info)¶

Displays the amount of time it took to load

(plugs.base.bookmark-clicked info bookmark)¶

Changes the URL when you click a bookmark

(plugs.base.return-url-bar info url)¶

Changes the current site to the site on the URL bar when you press enter.

(plugs.base.url-changed info url)¶

Changes the URL bar when the url of the webview changes.

(plugs.base.refresh info)¶

Handles refreshing when you press F5/The refresh button

(plugs.base.add-bookmark info url)¶

Adds a new bookmark!

(plugs.base.reloaded info)¶

Called when the plug is reloaded
